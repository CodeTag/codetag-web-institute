DEV CAMP

A developer bootcamp is an intense program where you start from the web programming basics to build a portfolio of projects using technologies like Python-Django, HTML5, CSS3, Postgresql, Mongodb, Javascript among others.

This is a hands-on bootcamp that require a lot of effort and dedication, beyond technologies you will be introduced to computer science subjects and practices.

THE PROGRAM

Methodology

The program is part online part onsite, however the time dedication/intensity is at least 8 hours a day, you will have two classes every day, also pairing time where you will be able to work in the weekly projects and receive feedback, collaboration

Contents

  • Typical Development enviroment
  • Web development - Front End
  • Web development - Back End
  • Deploying web applications
  • Testing - Debugging
  • Real time web
  • Resources Management
