DEV CAMP

Un campamento de desarrollo es un programa intenso donde empiezas con las bases del framework web Django, para seguir aprendiendo como desplegarlo y como esta construida una aplicacion con este framework.

Este es un campamento que requiere mucho esfuerzo y dedicación, mas que teoría seras introducido a las mejores practicas de este framework, su arquitectura, y sera guiado para construir 4 proyectos que lo llevaran a aprender de esta tecnología.
